# Release Notes

This document provides a summary of all releases and their key features.

## v3.3.0 - Cloud Integration (Q3 2026)

### Features
- Cloud Sync: Settings and data synchronization across devices
- Remote Monitoring: Web-based monitoring dashboard
- Mobile App: Companion app with push notifications
- API Access: REST and WebSocket APIs for external integrations

See [V3.3.0_FEATURES.md](V3.3.0_FEATURES.md) for details.

## v3.2.0 - Advanced Analytics (Q2 2026)

### Features
- Advanced Charts: Interactive TradingView-style charts
- Portfolio Analytics: Sharpe ratio, Sortino ratio, drawdown analysis
- Risk Analytics: VaR, CVaR, stress testing
- Market Depth: Level 2 order book visualization
- Correlation Analysis: Multi-currency correlation matrix
- Economic Calendar: Economic events integration
- Trade Journal: Automated trade journaling

See [V3.2.0_FEATURES.md](V3.2.0_FEATURES.md) for details.

## v3.1.0 - Enhanced Strategies (Q1 2026)

### Features
- ML Strategies: LSTM, Transformer, RL, SVM, Ensemble
- Strategy Optimization: Genetic algorithms, hyperparameter optimization
- Strategy Builder: Visual strategy builder with templates
- Strategy Marketplace: Community-driven strategy sharing
- Enhanced Backtesting: Parallel processing, improved reliability

See [V3.1.0_FEATURES.md](V3.1.0_FEATURES.md) for details.

## v3.0.0 - Core Platform

### Features
- Core trading engine
- MT4 integration
- Paper trading
- Basic strategies
- Risk management
- UI framework
